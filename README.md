 To run the backend code, you should
 
  Ensure you have nodejs version "16.18.0" and above installed.

   Install npm package
   
 

>   `npm i`

    node index.js

 Install nuxt 
 
 https://nuxtjs.org/docs/get-started/installation
 
  Install npm package
  
 

     npm i

  
   To run the frontend service, use the command below:
    
           npm run dev
